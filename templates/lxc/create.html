About New LXC Container

Downloaded Templates — choose from templates already stored on your Proxmox node. These are ready to use immediately with no download required.

Available to Download — browse the official Proxmox template catalogue. Selecting a template will download it to your storage before creating the container.

After selecting a template you'll configure the container — hostname, resources (CPU, RAM, swap), storage, networking, and authentication — before creation begins.

Template storage must have the vztmpl content type enabled in Proxmox. Go to Datacenter → Storage → select pool → Edit → Content.
