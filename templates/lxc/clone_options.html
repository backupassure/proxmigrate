About Container Clone

Cloning creates a full, independent copy of this container with a new CTID and hostname. The clone includes the complete rootfs and all configuration.

The cloned container is fully standalone — no dependency on the source. You can freely modify or delete either container afterwards.

The clone will inherit the source container's network config. You may need to update the IP address in Proxmox after cloning to avoid conflicts.
