VM Export

Export any Proxmox VM as a portable, self-contained package that can be shared, archived, or imported into another Proxmox server or cloud provider.

Portable Packages
Export VMs as versioned, compressed archives ready for long-term storage or sharing.
Cloud Export
Export directly to S3-compatible object storage (AWS S3, MinIO, Backblaze B2).
Snapshot Export
Export specific VM snapshots, not just the current state, for point-in-time backups.
Format Options
Export as qcow2, VMDK (for VMware), or raw format for maximum compatibility.
Encrypted Exports
Optional AES-256 encryption for exported packages — protect sensitive VM data at rest.
Round-Trip Import
Exported packages can be imported back into any ProxMigrate instance — full portability.
