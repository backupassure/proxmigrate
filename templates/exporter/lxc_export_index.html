About Container Packages

Export — packages a container's full configuration and rootfs into a portable .px file using vzdump snapshot mode (zero downtime).

Import — restores a container .px package onto any ProxMigrate server. You can review and adjust hostname, resources, network, and storage before deployment.

Use cases: migrate containers between Proxmox clusters, back up container configurations, or share development environments.

Completed exports are automatically deleted after 24 hours.
