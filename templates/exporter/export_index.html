About .px Packages

Export — packages a VM's full configuration and all disks into a portable .px file. Trigger an export from any VM's detail page.

Import — restores a .px package onto any ProxMigrate server. You can review and adjust all settings (name, memory, CPU, network, storage) before deployment.

Use cases: migrate VMs between Proxmox clusters, back up VM configurations, share development environments, or deploy templated VMs.

Completed exports are automatically deleted after 24 hours.
