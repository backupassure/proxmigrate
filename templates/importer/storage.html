How Cleanup Works

Automatic cleanup: When an import job completes successfully or fails, ProxMigrate automatically deletes the local upload file and any temporary files on the Proxmox staging directory.

What can remain on disk:

  • Jobs that were interrupted (e.g. server restarted mid-transfer)
  • Jobs still in progress
  • Files uploaded but never configured/submitted

Use Clean Up Completed & Failed to remove all local files for jobs that have finished (successfully or not). Files for jobs still in progress are left untouched.
