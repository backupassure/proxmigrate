Create API Token

ProxMigrate needs a Proxmox API token to manage VMs. Follow these steps in your Proxmox web interface.

{# Step-by-step guide #}
Steps in Proxmox Web UI
  1. 1
    Open your Proxmox web interface at https://{{ proxmox_host }}:{{ api_port }} and log in as root.
  2. 2
    Navigate to: Datacenter → Permissions → API Tokens → click Add.
  3. 3
    Set: User = root@pam, Token ID = proxmigrate, and uncheck "Privilege Separation" so the token inherits root permissions.
  4. 4
    Click Add. Proxmox will show the token secret once only. Copy it immediately — you cannot retrieve it again.
{# Warning about token secret #}
Copy the secret now. Proxmox only shows the API token secret once, immediately after creation. If you miss it, you'll need to delete the token and create a new one.
