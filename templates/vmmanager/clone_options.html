About VM Clone

Cloning creates a copy of this VM with a new VMID and name. A full clone is completely independent — you can modify or delete either VM freely.

A linked clone shares the source VM's base disk and uses less space, but depends on the source VM's disk. Use linked clones for temporary or test VMs.

The clone inherits the source VM's network config including MAC address. Proxmox will generate a new MAC automatically, but you may need to update any static IP configuration inside the guest OS.
