{# Azure App Registration guide #}
Azure App Registration Guide
  1. 1
    Go to portal.azure.com → Microsoft Entra ID → App registrations → New registration.
  2. 2
    Name: ProxMigrate. Supported account types: Accounts in this organizational directory only (or multi-tenant if needed). Add the Redirect URI shown above.
  3. 3
    After creation, copy the Application (client) ID and Directory (tenant) ID from the overview page.
  4. 4
    Go to Certificates & secrets → New client secret. Copy the secret Value (shown once only).
