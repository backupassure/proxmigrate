About New VM

Install from ISO — upload any bootable ISO (Linux, Windows, network OS, etc.). ProxMigrate transfers it to Proxmox ISO storage, creates the VM, attaches the ISO as a virtual CD-ROM, and boots from it. You complete the installation inside the VM console.

Blank VM — creates a fully configured VM shell with no media attached. Useful when you want to attach an existing Proxmox ISO or disk manually after creation.

Importing an existing disk image? Use Import Disk instead — it handles qcow2, vmdk, vhd, vhdx, raw, img, and OVA.

ISO storage must have the iso content type enabled in Proxmox. Go to Datacenter → Storage → select pool → Edit → Content.
